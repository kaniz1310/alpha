## Overview
Final.Kaniz is a responsive, multi-page corporate-style website built using HTML, CSS, JavaScript, and Bootstrap. The project emphasizes clean UI design, structured layout, and responsive behavior across devices. It was developed as part of an academic and portfolio project to demonstrate frontend development fundamentals.

## Live Preview
Open `index.html` in any modern web browser to view the website.

## Technologies
- HTML5  
- CSS3  
- JavaScript  
- Bootstrap 5  


## Features
- Fully responsive layout for desktop, tablet, and mobile devices  
- Modern and clean user interface  
- Multi-page navigation system  
- Reusable components and organized file structure  
- Cross-browser compatibility
